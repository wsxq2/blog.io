### 整数运算
1. `$(())`:
   ```
   $ echo "$((5*5+5-3/2))"
   29
   ```
2. `expr`:
   ```
   $ expr 5 - 4
   1
   ```
### 浮点运算
1. `bc`:
   ```
   $ echo "5.01-4*2.0"|bc
   -2.99
   ```
2. `awk`:
   ```
   $ awk 'BEGIN{print 7.01*5-4.01}'
   31.04
   ```
